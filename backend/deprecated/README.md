# Deprecated Files

This directory contains legacy files and scripts that are no longer used in the active application but are kept for reference.

- `app_legacy.py`: The original monolithic Flask application file before refactoring into the `app/` package.
- `kmeans_cluster.py`: Old clustering script, logic moved to `app/services/cluster_service.py`.
- `display_outbreak_results.py`: Standalone script for displaying outbreak results, not used by the API.
- `debug_lang_check.py`: Debug script for language detection.
